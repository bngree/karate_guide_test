Рэнраку

1.И.П. М.СНД. На один шаг в СНД:
ч.гяко-цуки + гяко учи-укэ + гяко уракен шомен ганмен-учи

2.И.П. Х.ККД: шуто маваши-укэ.На два шага:
джодан мае-гери(М.ККД-К) + Х.ККД: шуто маваши-укэ + д.моротэ-учи
поворот: ККД: шуто маваши-укэ + д.моротэ-учи

3.И.П. Х.ЗКД. На три шага:
первый: М.ККД: шуто маваши-укэ + М.ЗКД: д.гяко йонхон-нукитэ
второй: Х.ККД: шуто маваши-укэ + Х.ЗКД: гяко уракен шомен-учи
третий: М.ККД: шуто маваши-укэ + М.ЗКД: гяко мэ-цуки
поворот: Х.ККД: шуто маваши-укэ + Х.ЗКД: ч.моротэ йонхон нукитэ

4.И.П. Х.СНД. На три шага:
первый: ой гедан-барай + ой уракен саю-учи + гяко уракен хизо-учи
второй: ой джодан-укэ + ой уракен ороши-учи + гяко уракен шомен-учи
третий: ой сото-укэ + ой уракен йоко-учи + гяко уракен маваши-учи
поворот: ой учи-укэ + ой уракен шомен-учи + гяко уракен ороши-учи

5.И.П. Х.ЗКД-К. На три шага:
г.маваши-гери + кансецу-гери + ч.сокуто йоко-гери

6.И.П. Х.КМД. На три шага:
д.мае-гери + г.маваши-гери + ч.сокуто йоко-гери

7.И.П. Х.КБД90*-К. На три шага через МКД-УКД-МКД:
кансецу-гери + ч.сокуто йоко-гери + чокуто йоко-киагэ
