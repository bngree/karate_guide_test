1.И.П. Х.ККД. На один шаг:
шуто маваши-укэ + д.моротэ-учи

2.И.П. Х.ЗКД. На один шаг:
М.ЗКД: джодан-укэ + М.ККД: сото-укэ + М.КБД90*: гедан-барай + М.НАД: учи-укэ

3.И.П. Х.ККД. Шуто маваши-укэ. На три шага:
первый: М.ККД: ой джодан-укэ + М.ЗКД: гяко тэцуи ороши-учи
второй: Х.ККД: ой сото-укэ + Х.ЗКД: гяко тэцуи коми каме-учи
третий: М.ККД: ой учи-укэ + М.ЗКД: гяко тэцуи хизо-учи
поворот: Х.ККД: шуто маваши-укэ + Х.ЗКД: ой д.йоко тэцуи-учи

4.И.П. Х.ЗКД-К. На три шага:
мае-киагэ + учи маваши-киагэ + сото маваши-киагэ

5.И.П. Х.ЗКД-К. На три шага:
первый: М.мае-киагэ(М.НАД) + М.ой учи маваши-киагэ(М.ЗКД)
второй: Х.учи маваши-киагэ(Х.НАД) + Х.ой сото маваши-киагэ(Х.ЗКД)
третий: М.сото маваши-киагэ(М.НАД) + Х.ой йоко-киагэ(М.ЗКД)

6.И.П. Х.ЗКД-К. На один шаг йонхон кери-ваза:
мае-киагэ + учи маваши-киагэ + сото маваши киагэ + йоко-киагэ

7.И.П. Х.ЗКД. На три шага:
первый:М.ККД: шуто маваши-укэ + М.КБД90*: д.тэцуи йоко-учи
второй:Х.ККД: шуто маваши-укэ + Х.КБД90*: ч.тэцуи йоко-учи
третий:М.ККД: шуто маваши-укэ + М.КБД90*: г.тэцуи йоко-учи
поворот:Х.КБД90*: моротэ гедан-барай
