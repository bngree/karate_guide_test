Рэнраку

1.И.П.М.СНД. На один шаг в СНД:
ч.мае-гери + ч.гяко-цуки + ч.учи-укэ + гяко уракен шомен ганмен-учи + гяко д.хиджи-атэ + гяко д.тэцуи йоко-учи

2.И.П. Х.ККД. На три шага в ККД:
первый:М.ККД: ой д.кокен-укэ + гяко ч.маваши кокен-учи
второй:Х.ККД: ой ч.кокен-укэ + гяко г.кокен-учи
третий:М.ККД: ой. г.кокен-укэ + гяко д.маваши кокен-учи
поворот:Х.ККД: ой ч.учи кокен-укэ + шомен ганмен кокен-учи

3.И.П. Х.ЗКД. На три шага:
первый:М.ККД: ой д.кокен-укэ + М.ЗКД: гяко ч.хиракен-цуки
второй:Х.ККД: ой ч.шуто-укэ + Х.ЗКД: гяко д.маваши хиракен-цуки
третий:М.ККД: ой г.шотэй-укэ + М.ЗКД: гяко д.ороши хиракен-цуки
поворот:Х.ККД: ой ч.хайшу-укэ + Х.ЗКД: гяко д.хиракен-цуки

4.И.П. Х.НАД. На три шага в НАД:
первый:М.НАД: ой д.кокен-укэ + ой г.шотэй-укэ +ой д.хиракен-цуки
второй:Х.НАД: ой г.шотэй-укэ + ой д.кокен-укэ + шуто сакотсу учи-коми
третий:М.НАД: ой ч.шотэй-укэ + ой ч.учи кокен-укэ + ой д.маваши хиракен-цуки
поворот:Х.НАД: ой ч.учи кокен-укэ + ой ч.шотэй-укэ + ой д.шуто учи-учи

5.И.П.Х.КМД. Нихон тоби кери-ваза на три шага в КМД:
ой тоби-гери + гяко тоби-гери + тоби хиза\мае-гери

6.И.П. Х.КМД. Нихон тоби кери-ваза на три шага в КМД:
тоби нидан-гери (чудан\чудан) + тоби нидан-гери(чудан\джодан) + тоби нидан-гери (джодан\джодан)

7.И.П. Х.ЗКД. Санбон цуки в ЗКД на один шаг передвигаясь в ура:
д.гяко-цуки + ч.ой-цуки + ч.гяко-цуки
