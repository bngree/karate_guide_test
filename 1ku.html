Рэнраку

1.И.П. Х.ЗКД. На три шага:
первый:М.ККД: ой д.каке-укэ + М.ЗКД: гяко д.накаюби иппонкен
второй:Х.ККД: ой д.хайто-укэ + Х.ЗКД: гяко д.рютокен-цуки
третий:М.ККД: ой ч.каке-укэ + М.ЗКД: гяко д.хитасашиюби иппонкен
поворот:Х.ККД: ой ч.хайто-укэ + Х.ЗКД: гяко д.ояюби иппонкен

2.И.П. Х.НАД. На три шага в НАД:
первый:ой д.накаюби иппонкен + ой ч.каке-укэ + ой д.иппон-нукитэ
второй:ой д.рютокен + ой д.хайто-укэ + ой д.нихонкен
третий:ой д.хитасашиюби иппонкен + ой д.каке-укэ + ой д.нихон нукитэ
поворот:г.ояюби иппонкен + ой ч.хайто-укэ + ой д.рютокен-цуки

3.И.П. Х.КМД. Нихон кери ваза одной ногой на три шага:
первый:ой учи хайсоку-гери (Х.КМД) + ой ороши учи какато-гери (движение ой-аши в Х.КМД)
второй:ой учи хайсоку-гери (Х.КМД) + ой ороши сото какато-гери (движение ой-аши в Х.КМД)
третий:ой учи хайсоку-гери (Х.КМД) + ой тоби йоко-гери (движение ой-аши в Х.КМД)

4.И.П. Х.КБД90*-К. Санбон кери-ваза одной ногой в КБД90*-К через МКД/УКД/МКД на три шага:
йоко кеагэ/кансетсу-гери/д.тоби йоко-гери + йоко кеагэ/ч.йоко-гери + йоко кеагэ/й.йоко-гери/д.тоби йоко-гери

5.И.П.Х.ЗКД. Передвижение в ЗКД на три шага в ура:
первый:ой джодан-укэ + ч.гяку-цуки
второй:ой сото-укэ + ч.гяко-цуки
третий:ой учи-укэ + ч.гяко-цуки
поворот:ой гедан-барай + ч.гяко-цуки
