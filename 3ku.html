1.И.П. Х.КБД45*-К Йонхон кери-ваза одной рукой на один шаг
д.мае-гери(45*) + д.йоко-гери(45*) + д.уширо-гери(45*) + д.маваши-гери(45*)

2.И.П. Х.ЗКД. На три шага:
первый:М.ККД: ой д.шотэй-укэ + М.ЗКД: гяко д.агэ-хиджи-атэ
второй:Х.ККД: ой ч.шотэй-укэ + Х.ЗКД: гяко д.хиджи-атэ
третий:М.ККД: ой. г.шотэй-укэ + М.ККД: гяко д.мае хиджи-атэ
поворот:Х.ККД: ой шуто гедан-барай + Х.ЗКД: ой уширо хиджи-атэ

3.И.П. Х.СНД. На три шага в СНД:
первый: ой джодан-укэ + ой ороши хиджи-атэ + гяко ч.аге хиджи-атэ
второй: ой сото-укэ + ой д.хиджи-атэ + гяко ч.мае хиджи-атэ
третий: ой учи-укэ + ой д.мае хиджи-атэ + гяко ч.хиджи-атэ
поворот: ой гедан-барай + ой д.агэ хиджи-атэ + гяко уширо хиджи-атэ

4.И.П. Х.ККД. Повторные блоки на три шага в ККД:
Джодан-укэ/гедан-барай + Сото-укэ/гедан-барай + Учи-укэ/гедан-барай
поворот: Х.КБД90*: Гедан-барай/Джодан учи-укэ

5.И.П.Х.ККД. Нихон кери-ваза на три шага в ККД:
первый:ой д.мае-гери (Х.ККД) + гяко г.какато мае-гери (М.ККД)
второй:ой ч.мае-гери (М.ККД) + гяко ч.какато мае-гери (Х.ККД)
третий:ой кин-гери (Х.ККД) + гяко д.какато мае-гери (М.ККД)

6.И.П. Х.ЗКД45*-К. На три шага:
первый:М.мае-гери Х.45*(Х.ЗКД45*) + М.45*/М.д.йоко уракен-учи(М.ЗКД45*) + Х.ч.гяко-цуки
второй:ч.мае-гери М.45*(М.ЗКД45*) + Х.йоко-гери Х.45*/Х.д.йоко уракен-учи(Х.ЗКД45*) + М.д.гяко хиджи-атэ (в ладонь передней руки)
третий: М.мае-гери Х.45*(Х.ЗКД45*) + М.йоко-гери М.45*/М.д.йоко-уракен-учи(М.ЗКД45*) + М.ой д.хиджи-атэ (усиленный)

7.И.П. Х.ЗКД. На три шага в ЗКД:
первый:д.мае-гери (М.ЗКД) + ч.ой-цуки
второй:д.маваши-гери (Х.ЗКД) + ч.гяко-цуки
третий:д.йоко-гери (М.КБД90*) + ч.дзюн-цуки
поворот:Х.КБД90*: моротэ гедан-барай
