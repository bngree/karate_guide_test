Рэнраку

1.И.П. М.СНД. На один шаг в СНД:
ч.гяко-цуки + гяко учи-укэ + гяко уракен шомен ганмен-учи + гяко д.хиджи-атэ

2.И.П. Нихон кери-ваза на один шаг одной ногой в КБД45*-К:
д.мае-гери (45*) + ч.йоко-гери (45*)

3.И.П. Х.ЗКД. На три шага:
первый: М.ККД: ой д.шотэй-укэ + ой д.хиджи-атэ + М.ЗКД: д.шотэй-учи
второй: Х.ККД: ой ч.шотэй-укэ + ой д.хиджи-атэ + Х.ЗКД: ч.шотэй-учи
третий: М.ККД: ой г.шотэй-укэ + ой д.хиджи-атэ + М.ЗКД: г.шотэй-учи
поворот: Х.ЗКД: г.джуджи-укэ + д.джуджи-укэ

4.И.П. Х.МАД. На три шага в МАД (фуми-аши):
первый: М.МАД: ой д.шотэй-укэ + гяко ч.шотэй-учи
второй: М.МАД: ой ч.шотэй-укэ + гяко г.шотэй-учи
третий: М.МАД: ой г.шотэй-укэ + гяко д.шотэй-учи
поворот: ой учи-укэ + гяко шита-цуки

5.И.П. Х.КМД. На три шага:
д.мае-гери + ч.уширо-гери + ч.маваши-гери чусоку

6.И.П. Х.КБД45*. На один шаг:
гяко д.татэ йонхон-нукитэ + ой д.хиджи-атэ (в ладонь) + ой уракен -гяко теншо-укэ (каке-укэ) + ой гедан-барай + ч.гяко-цуки

7.И.П. Х.ЗКД-К. На три шага:
первый: М.ч.уширо-гери 180* (М.УЗКД-К)
второй: УКД:М.ч.уширо-гери (Х.УЗКД-К)
третий: Х.ч.уширо-гери (Х.УЗКД-К)
поворот: М.ЗКД-К
